
# Card Outcome Probability Calculator

## Project Description
A C++ console-based project that calculates theoretical and experimental probabilities
of drawing cards from a standard 52-card deck using Discrete Mathematics concepts.

## Folder Structure
- src/   -> Source code files
- test/  -> Test cases
- data/  -> Sample input data

## Installation
Compile using any C++11 compatible compiler.

## Usage
Run the executable and choose options from the menu.

## Dependencies
Standard C++ libraries only.
