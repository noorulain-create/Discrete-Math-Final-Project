# Card Outcome Probability Calculator
## Overview
The **Card Outcome Probability Calculator** is a menu-based C++ program developed to calculate probabilities related to drawing cards from a standard 52-card deck.  
It applies concepts of **Discrete Mathematics**, including probability, counting principles, combinations, and Monte Carlo simulation.
## Project Track
**Track 6: Counting & Combinatorics**
## Features

### Two-Card Probability
- Probability of two cards having the same rank  
- Probability of drawing one Ace and one non-Ace
